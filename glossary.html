Hyper Text Markup Language Revision 5 (HTML5) is a markup language used to structure and present the content of World Wide Web. HTML5 supports the conventional HTML and XHTML-style syntax and other new features in it are markup, new APIs, XHTML and error handling

Cascading Style Sheet (CSS) is a standard language that traces the format of markup language pages. CSS helps developers to divide content and visualize elements for great page control and flexibility. A CSS file is usually attached to an HTML file by means of link in the HTML file.

Java Script (JS) is a scripting language used on the web which is used to reinforce html pages and commonly found attached in html codes. Java script is interpreted language. Thus the compiling is not necessary. Java script provides webpages on interactive and dynamic fashion. This Allows pages to react to events, exhibit, validate data and create cookies, etc.

The CSS grid layout shines at dividing a page into major regions and minor regions or by defining the relationship in terms of position, size, layers , and other between parts of a control built primitives

CSS Animation makes it achievable to animate transion form one CSS style configuration to other. Animation is composed of two components a style expressing CSS animations and set of keyframes that shows the start and end state of animation style.

The Box Shadow is the property of CSS which adds shadow effect around the elements frame. Multiple Box shadow effect can be set using separated commas. A box shadow is defined by X and Y offsets relative.
